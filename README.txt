解ける問題
        9x9普通のやつ
	nxn普通のやつ (n = 1...25)
        不等号
        対角線: まだ
        一差: まだ
        ツートン: まだ
        あろー: まだ
        クク: まだ
        奇数偶数: まだ
        組み合わせ: 結構面倒くさい

操作
	数字の入力: カーソルか矢印キーで四角を動かしてキーボードで数字を打つ
		(0, space, backspace を打つと数字がなくなる)
	戻る: cmd z  (不等号の戻るはまだできない)(戻ったら進めない)
	不等号を入れる: alt ←↓↑→
	全部消す: n
	マス数を変える: cmd n を押して数字を入力して enter (1~25マスまで)
